# 用户生命周期

固定阶段：

`无意识 -> 有意识 -> 记住你 -> 关注你 -> 了解你 -> 接触你 -> 好朋友 -> 选择你 -> 有收获 -> 转介绍`

每个阶段必须填写：

- 阶段目标
- 典型场景
- 接触点
- 用户想法
- 用户行为
- 核心痛点
- 信任障碍
- 推进动作
- 判断信号

注意：

`核心痛点` 每个阶段至少 3 条，优先 3-5 条。
