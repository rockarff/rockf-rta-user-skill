# Interview Paths

## Zero-Based Path

Use this when the client has not built enough real customer evidence yet.

Modules:

1. personal background and career path
2. people they naturally help well
3. problems they understand deeply
4. natural speaking and judgment style
5. preferred future customer
6. customer they do not want

Goal:

Build a first working hypothesis for:

- likely user groups
- likely trust path
- likely persona direction

## Validated Path

Use this when the client has enough customer evidence.

Modules:

1. current offer and business structure
2. recent paying customers
3. best-fit customers
4. worst-fit customers
5. trust and buying triggers
6. result, repeat purchase, referral
7. existing content feedback

Goal:

Extract:

- real user groups
- real decision barriers
- real trust logic
- real conversion patterns

## Mixed Path

Use this when the client has some evidence, but not enough to rely on only the past.

Modules:

1. current business stage
2. small set of real customers
3. strongest problem-solving zone
4. preferred future customer
5. natural expression style
6. trust-building style
7. early content or market feedback

Goal:

Separate:

- what is already verified
- what is still a hypothesis
