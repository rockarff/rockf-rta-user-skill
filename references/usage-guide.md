# Usage Guide

## Recommended default route

Use the full route when the client is new or the materials are incomplete:

`Discovery -> User -> Persona -> Content -> Workflow -> Report`

## Entry decision

### Start from Discovery when:

- client type is still unclear
- no clear transcript exists
- no clear near-close portrait exists
- no clear mother-topic seed exists

### Start from User when:

- a usable interview transcript already exists
- Discovery judgment is already stable

### Start from Persona when:

- user profiles and lifecycle are already stable

### Start from Content when:

- persona has already been judged
- mother-topic direction is already usable

### Start from Workflow when:

- the operator is not sure what is missing
- the operator needs a current-stage judgment
- the operator wants one unified handoff package

## What to protect at each stage

### Discovery

- client type
- near-close portrait
- unfit portrait
- mother-topic seeds

### User

- portrait groups
- ten-stage lifecycle

### Persona

- native mother topics
- assisted mother topics
- disallowed topics
- expression red lines

### Content

- topic source
- portrait basis
- topic structure

### Workflow

- gate status
- blockers
- next step

### Report

- only final conclusions
- not raw working notes
