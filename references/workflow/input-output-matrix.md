# Input Output Matrix

## Discovery

Input:

- client basics
- current stage
- existing proof or no proof

Output:

- client type
- interview script
- material checklist
- downstream-customer portrait collection plan
- mother-topic seed collection plan

## User

Input:

- transcript or interview answers

Output:

- user profiles
- user lifecycle
- portrait split:
  - closed
  - near-close
  - unfit

## Persona

Input:

- transcript
- user profiles
- user lifecycle

Output:

- persona
- expression guide
- native / assisted / disallowed mother topics
- expression red lines

## Content

Input:

- user profiles
- user lifecycle
- persona
- portrait split from upstream

Output:

- content judgment
- content map
- topic pool
- script or interview guide

## Workflow

Input:

- discovery output
- user output
- persona output
- content output

Output:

- workflow status
- workflow handoff
- normalized report input
