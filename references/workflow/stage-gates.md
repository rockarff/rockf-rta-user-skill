# Stage Gates

Gate 1:

- Discovery done
- human confirms client type and interview path
- downstream-customer portrait collection path is clear:
  - closed
  - near-close
  - unfit
- mother-topic seed path is clear:
  - repeated judgment topics
  - repeated misjudgment topics
  - strongest scenes
  - strongest stances

Gate 2:

- user profiles and lifecycle done
- human confirms they are usable
- portrait split is usable for later broad-traffic / semi-vertical targeting
- ready-to-convert portraits are concrete enough to support topic targeting

Gate 3:

- persona done
- human confirms expression direction
- human confirms native mother topics
- human confirms expression red lines

Gate 4:

- topic pool done
- human confirms selected topics before script production
