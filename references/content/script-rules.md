# 脚本规则

## 如果是自拍脚本

至少要有：

1. 开头钩子
2. 中段结构
3. 结尾钩子
4. 评论区引导

可选补充：

- 镜头提示
- 节奏提示
- 关键词强调

## 如果是访谈拍摄稿

至少要有：

1. 访谈目标
2. 主问题
3. 追问问题
4. 希望现场采到的原话
5. 后期可剪方向

## 生成规则

1. 不要一上来写空泛观点。
2. 优先用用户阶段和内容目标来定表达。
3. 一个稿件只服务一个主目标，不要混太多目的。
4. 批量辅助框架默认按表格输出。

## 辅助框架表格结构

如果用户要求为已选中的题批量出辅助框架，默认使用横向表格。

列顺序固定为：

1. 编号
2. 选题
3. 开头钩子
4. 中段结构
5. 结尾钩子
6. 评论区引导
7. 完整脚本
