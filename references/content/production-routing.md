# 生产分流规则

RTA-CONTENT 在进入生产前，必须先判断输入方式。

只允许两种：

- `self_shot`
- `interview_shot`

## self_shot

适合：

- 客户自己讲
- 主理人自己口播
- 观点、判断、拆解型内容

要输出：

- 开头钩子
- 结构提纲
- 结尾钩子
- 评论区引导
- 可选完整逐字稿

## interview_shot

适合：

- 主理人被采访
- 团队成员访谈主理人
- 客户案例型、经历型、故事型内容

要输出：

- 访谈目标
- 主问题
- 追问问题
- 素材采集目标
- 后期可剪方向

## 禁止事项

1. 未判断输入方式前，不要直接出脚本。
2. `interview_shot` 模式下，不要默认输出口播逐字稿。
3. `self_shot` 模式下，不要只给问题，不给表达结构。
4. 未经过选题确认前，不要默认批量输出完整脚本。
