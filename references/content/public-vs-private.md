# 公域与私域分工规则

## 公域优先

更适合做公域传播的，通常是：

- 潜在用户
- 核心用户
- 用户生命旅程 1-6 阶段
- 默认公域结构：广流量 / 半垂直流量 / 信任 / 转化

原因：

- 更容易被短视频承接
- 更适合做风险识别、代入、判断、咨询触发

## 私域优先

更适合做私域内容的，通常是：

- 高价值用户
- 生命周期 3-10 阶段中的关系维护部分

原因：

- 重点不是停留，而是长期信任
- 重点不是第一次咨询，而是长期合作和转介绍

## 强规则

1. 不要把高价值用户默认当成公域短视频主战场。
2. 不要把 7-10 阶段默认当成短视频主选题区。
3. 公域内容主打识别、代入、信任、咨询触发。
4. 私域内容主打关系、合作、复用、转介绍。
5. `private_domain` 不再默认进入公域内容配比，只在明确要求时单独生成。
