# Output Structure

Markdown output should include:

1. 核心人设句
2. 一句话价值
3. 适配用户
4. 信任角色
5. 表达风格
6. 内容主轴
7. 信任建立方式
8. 转化姿态
9. 表达禁区
10. 后续内容使用建议

JSON output should include these top-level keys:

- `persona_core`
- `value_proposition`
- `fit_users`
- `trust_role`
- `expression_style`
- `topic_axes`
- `trust_mechanism`
- `conversion_posture`
- `red_lines`
- `content_guidance`
- `confidence_notes`
